
For non-empty directories, use the `rm` command with the `-r` (recursive) option:

- `rm -r directory_name` deletes the directory and all its contents recursively
- In Linux or Unix-like systems, the `touch` command creates an empty file. If the directory does not exist, first use `mkdir` to create it. For example, to create a directory named `mydir` and a file named `myfile.txt` inside it, run:

```
mkdir mydir && touch mydir/myfile.txt
```
